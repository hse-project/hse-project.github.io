    Resources

    Resources are available to report a bug, ask a question, or provide feedback on how we can improve HSE or this documentation.

    Report a Bug

    You can file an issue to report an HSE bug. When reporting a bug, please attach the output of the following command.

    $ sudo sosreport --batch

    If you encounter issues generating a complete report, please try the following.

    $ sudo sosreport -o hse

    Ask a Question

    You can ask a question in the HSE discussions forum using the Q&A category.

    Participate in Discussions

    You can discuss all aspects of HSE in the discussions forum using the following categories.

    • Q&A to ask a question
    • Ideas to recommend a feature or enhancement
    • Show and tell to talk about your experiences with HSE
    • General to discuss all other aspects of the project
